## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/KingsleyAmankwah/rooms-homepage

# Access
$ cd rooms-homepage

# Install dependencies
$ npm install

# Run the project
$ npm start

# The server will initialize in the <http://localhost:3000>
```
